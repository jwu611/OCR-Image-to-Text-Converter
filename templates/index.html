OCR Image to Text Converter

This application uses Python's OpenCV library to process an image and a Pytorch machine learning classification model to identify the letters in the image and output them in text formatted similarly to the image.

Due to lack of available training datasets, the app is only able to recognize capital letters (A-Z) and accuracy may vary if the lettering is too thin (the ideal letter thickness is with a medium sharpie like the default image).
The app also assumes that the uploaded image only contains text.

More Information

The image preprocessing involves grayscaling, Gaussian blurring, and finally thresholding to reduce noise. The classifier uses transfer learning with a pretrained Resnet34 Convolutional Neural Network.

For more information on the preprocessing, how the model was trained and accuracy was improved, please visit the project page at jwu611.github.io
